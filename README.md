# jimbooth.ai

Personal site for Jim Booth - Technical Sales Leader, AI Infrastructure. Migrated from the previous endur.dev deployment.

## Structure
- index.html - homepage
- posts/ - articles (served at /posts/<slug> via cleanUrls)
- *.svg, *.png - figures and screenshots
- Jim_Booth_Resume.pdf - resume
- favicons + site.webmanifest

## Hosting
Static site on Vercel (no build step). vercel.json enables clean URLs.
